About Our Project

Our challenge was, "Hey! What Are You Looking At?" To complete the challenge, we decided to create a web-based application that would allow anybody interested in astronomy to easily view astronomical observations, and encourage them to learn more about astronomy. A user can interact with the "sky sphere" at the top of the page, and click on observations they'd like to learn more about. A card below the map displays some informaion about the observation, and provides a download link for the FITS image file.

Below the map is a list of all retrieved objects, and buttons to download their FITS image.

The last feature is a SOMETHING.

How It Works

This project uses Flask, a micro web framework written for Python. We manually selected five CSA NEOSSat observations, which are then retrieved within Python, converted into dictionaries, and passed into the HTML template. Inside the HTML webpage, the data is processed, and placed into the "sky sphere" and the list. The "sky sphere" is a JavaScript sky map created by Sonia Zorba, and can be found at http://zonia3000.github.io/SkySphere.
